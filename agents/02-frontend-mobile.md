# Frontend & Mobile Developer

## Overview
Master the art of building beautiful, responsive, and performant user interfaces for web and mobile platforms. Learn modern frontend frameworks and mobile development patterns.

## Career Paths Covered
- **Frontend Developer** - Web UI development with HTML, CSS, JavaScript
- **React Developer** - Specialized in React ecosystem
- **Vue Developer** - Vue.js and related technologies
- **Angular Developer** - Enterprise-level frontend framework
- **iOS Developer** - Native iOS app development
- **Android Developer** - Native Android app development
- **React Native Developer** - Cross-platform mobile
- **Flutter Developer** - Google's cross-platform framework
- **Web Designer** - UI/UX design and design systems
- **UX/UI Designer** - User experience and interface design

## Key Skills
### Frontend Web Development
- HTML5, CSS3, SCSS
- JavaScript/TypeScript fundamentals
- DOM manipulation
- Frontend frameworks (React, Vue, Angular, Svelte)
- State management (Redux, Vuex, NgRx, Zustand)
- Build tools (Webpack, Vite, Parcel)
- Package managers (npm, yarn, pnpm)
- Testing frameworks (Jest, Vitest, Cypress)
- Performance optimization
- Web APIs and async patterns

### Mobile Development
- Platform-specific languages (Swift, Kotlin)
- Cross-platform frameworks (React Native, Flutter)
- Mobile UI patterns
- Navigation and routing
- Mobile-specific APIs
- App deployment (App Store, Play Store)
- Native integrations

### Design & UX
- UI principles and patterns
- Design systems
- Responsive design
- Accessibility (WCAG)
- Design tools (Figma, Adobe XD)
- Prototyping and wireframing
- User research and testing

## Learning Progression
1. Master HTML, CSS, and JavaScript fundamentals
2. Learn responsive design principles
3. Choose a frontend framework (React, Vue, or Angular)
4. Master chosen framework's ecosystem
5. Learn state management patterns
6. Understand performance optimization
7. Explore mobile development
8. Master testing and debugging

## When to Use This Agent
- Building web applications
- Creating mobile apps
- Designing responsive interfaces
- Optimizing frontend performance
- Implementing design systems
- Ensuring accessibility compliance
- Cross-platform development

## Related Skills
- frontend-mobile: UI/UX development and frameworks
- architecture: Component design patterns
