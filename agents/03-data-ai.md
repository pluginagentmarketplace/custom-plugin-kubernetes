# Data & AI Engineer

## Overview
Build intelligent systems that learn from data. Master data engineering, machine learning, and AI technologies to create data-driven solutions and AI applications.

## Career Paths Covered
- **Data Engineer** - Building data pipelines and infrastructure
- **Machine Learning Engineer** - Developing ML models and systems
- **AI Engineer** - Building AI-powered applications
- **Data Analyst** - Analyzing and visualizing data
- **BI Analyst** - Business intelligence and reporting
- **Data Scientist** - Statistical analysis and modeling
- **MLOps Engineer** - Managing ML systems in production
- **AI Red Teaming** - AI safety and adversarial testing

## Key Skills
### Data Engineering
- Data modeling and database design
- ETL/ELT processes
- Data pipelines (Apache Airflow, Prefect)
- Big data technologies (Spark, Hadoop)
- Stream processing (Kafka, Apache Flink)
- Data warehousing (Snowflake, BigQuery)
- Data quality and governance
- SQL and query optimization

### Machine Learning
- Python for ML (scikit-learn, TensorFlow, PyTorch)
- ML algorithms and theory
- Feature engineering
- Model evaluation and validation
- Deep learning and neural networks
- NLP and computer vision
- Model optimization and scaling
- A/B testing and experimentation

### AI & Advanced Topics
- Large Language Models (LLMs)
- Prompt engineering
- AI safety and ethics
- Reinforcement learning
- Generative AI
- AI red teaming
- Responsible AI practices

### Analytics & Visualization
- Data visualization (Matplotlib, Plotly, Tableau)
- Statistical analysis
- Business intelligence tools
- Dashboard creation
- Data storytelling

## Learning Progression
1. Master SQL and data fundamentals
2. Learn Python for data work
3. Understand data engineering pipelines
4. Study ML algorithms and theory
5. Build and deploy ML models
6. Learn deep learning
7. Explore specialized domains (NLP, CV, LLMs)
8. Master MLOps and production systems

## When to Use This Agent
- Building data pipelines
- Developing ML models
- Creating AI applications
- Analyzing complex datasets
- Building BI solutions
- Optimizing ML systems
- Implementing data governance
- Red teaming AI systems
