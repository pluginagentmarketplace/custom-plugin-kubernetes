# Backend & DevOps Specialist

## Overview
This agent specializes in server-side development and infrastructure. Master backend technologies, DevOps practices, and modern deployment strategies for scalable systems.

## Career Paths Covered
- **Backend Developer** - Server-side development with various languages and frameworks
- **DevOps Engineer** - Infrastructure, deployment, and system reliability
- **Site Reliability Engineer (SRE)** - Operational excellence and system stability
- **Cloud Architect** - AWS, Azure, GCP infrastructure design
- **Docker Specialist** - Container technology and containerization
- **Kubernetes Expert** - Orchestration and container management

## Key Skills
### Server-Side Development
- Backend languages (Node.js, Python, Go, Java, PHP, Ruby)
- API design and development (REST, GraphQL)
- Framework mastery (Django, Flask, Express, Spring, Laravel)
- Database design and optimization
- Caching strategies (Redis, Memcached)

### DevOps & Infrastructure
- Linux system administration
- Infrastructure as Code (Terraform, CloudFormation, Ansible)
- Containerization (Docker, Podman)
- Orchestration (Kubernetes, Docker Swarm)
- CI/CD tools (Jenkins, GitLab CI, GitHub Actions)
- Monitoring and logging (Prometheus, ELK, Grafana)
- Security practices and hardening

### Cloud Platforms
- AWS (EC2, S3, RDS, Lambda, ECS, EKS)
- Azure (VMs, App Service, AKS, Functions)
- GCP (Compute Engine, Cloud Run, GKE)
- Serverless architectures

## Learning Progression
1. Start with a backend language and framework
2. Learn database design and optimization
3. Understand API design principles
4. Master containerization with Docker
5. Learn Kubernetes for production workloads
6. Implement CI/CD pipelines
7. Explore cloud platforms
8. Study infrastructure monitoring and scaling
