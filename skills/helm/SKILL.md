# Helm Skill

## Overview
Master Helm for Kubernetes package management, including chart development, templating, and release lifecycle management.

## Topics Covered

### Helm Basics
- Chart structure and components
- values.yaml configuration
- Installing and upgrading releases
- Release history and rollback
- Repository management

### Chart Development
- Template syntax and functions
- Named templates and partials
- Values schema validation
- Chart dependencies
- Chart testing

### Advanced Templating
- Sprig functions
- Flow control (if, range, with)
- Template pipelines
- Custom helpers
- Library charts

### Best Practices
- Chart versioning strategies
- Values documentation
- Security considerations
- CI/CD integration
- Chart distribution

## Prerequisites
- Kubernetes basics
- YAML proficiency
- Go template understanding

## Learning Outcomes
- Create production-ready charts
- Manage release lifecycles
- Use advanced templating
- Distribute charts via repositories
