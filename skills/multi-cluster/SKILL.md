# Multi-Cluster Skill

## Overview
Manage multiple Kubernetes clusters for high availability, disaster recovery, and global deployments.

## Topics Covered

### Multi-Cluster Architecture
- Hub-spoke patterns
- Federated deployments
- Global load balancing
- Cross-cluster networking
- Cluster topology design

### Management Tools
- Rancher multi-cluster
- Red Hat ACM
- Kubernetes Federation
- Loft for virtual clusters
- Crossplane control planes

### Networking
- Multi-cluster service discovery
- Submariner setup
- Cilium Cluster Mesh
- Global DNS strategies
- Cross-cluster mTLS

### GitOps Multi-Cluster
- Fleet management
- ArgoCD ApplicationSets
- Cluster generators
- Policy distribution
- Configuration drift detection

### Disaster Recovery
- Active-active setups
- Active-passive failover
- Data replication
- RTO/RPO planning
- Backup strategies

## Prerequisites
- Single cluster mastery
- Networking expertise
- GitOps understanding

## Learning Outcomes
- Design multi-cluster architectures
- Implement cluster federation
- Configure global load balancing
- Set up disaster recovery
