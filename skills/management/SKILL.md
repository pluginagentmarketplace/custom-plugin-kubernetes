# Management & Product Skills

## Quick Start

### OKR Framework Example
```
Objective: Build market-leading user authentication system

Key Results:
1. Reduce authentication latency to <50ms (from current 200ms)
2. Achieve 99.99% uptime for auth services
3. Increase enterprise customer adoption by 40%
4. Complete security audit with zero critical findings

Initiatives:
- Optimize database queries
- Implement distributed caching
- Conduct security assessment
```

### 1-on-1 Meeting Template
```markdown
## Weekly 1-on-1 with [Team Member]

### Updates from last week
- Completed X task
- Blocked by Y issue

### This week's goals
- Goal 1
- Goal 2

### Feedback
- What went well?
- What could improve?

### Career growth
- Learning goals
- Skill development
```
